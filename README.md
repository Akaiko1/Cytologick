# DemetraAI

## Installation instructions

**Windows 10**

1. Install **python3**, version **3.10.6** or later

2. Create and activate new venv with **cmd** or **powershell** in local folder

3. Run command: 
   
   ```
   pip install -r requirements.txt
   ```

4. Also run:
   
   ```
   pip install -q git+https://github.com/tensorflow/examples.git
   ```

5. Modify setting at config.py

6. Start GUI with:
   
   ```
   python start.py
   ```
